# DoubleU Official | Artist Platform
**The digital hub for DoubleU — Cinematic Soul-Trap from the 405.**

This platform is a high-performance, mobile-responsive web application built to showcase the discography, media assets, and branding of music artist DoubleU. It utilizes a dynamic "Site Engine" that allows for content management directly via Supabase.

## 🚀 Tech Stack
* **Frontend:** HTML5, Tailwind CSS (Mobile-First)
* **Backend:** Supabase (PostgreSQL, Real-time)
* **Deployment:** Vercel (Clean URLs & Global Edge Network)
* **Logic:** Vanilla JavaScript / Supabase JS Client
